# API Design
The `Fetch` virtual table maps SQL queries to [HTTP Requests](https://developer.mozilla.org/en-US/docs/Web/HTTP/Guides/Messages#http_requests).
| SQL Query | HTTP Method |
|-----------|-------------|
| `SELECT`    | GET         |
| `INSERT`    | POST        |
| `UPDATE`    | PUT         |
| `DELETE`    | DELETE |
