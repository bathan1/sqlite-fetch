◆ str

typedef char str

Flattened struct type that writes sizeof(size_t) SIZE bytes at the head, and then has SIZE + 1 bytes after for the string + nul terminator.

For example, "hi" on big endian on most 64 bit machines would be encoded as:

|-—0--—|-—1-—| ... |-—7-—|-—8-—|-—9-—|-—10-—| | 0 | 0 | ... | 2 | 'h' | 'i' | 0 |
